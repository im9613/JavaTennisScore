테니스 계수기 설명서

0. 번 입력으로 프로그램 안에서 메뉴별 사용 설명서를 확인할 수 있습니다.

1.  새 경기 시작
	- 새로운 경기를 생성합니다. 경기 생성은 사전에 등록되어있는 선수들로만 생성이 가능합니다.
	- 동성간 경기만 가능합니다! 
	- 남성부 경기는 5판 3선승제, 여성부 경기는 3판 2선승제로 진행됩니다.
	- 모든 경기에는 듀스, 타이브레이크가 적용됩니다.
	- 경기가 끝난 후 스코어 보드로 세트스코어가 출력되며 저장됩니다. 
	- 저장된 경기내용은 2. 지난 경기 불러오기에서 확인 가능합니다.

2. 지난 경기 불러오기
	- [1. 새 경기 시작]을 통해 기록된 경기를 불러옵니다.
	-  리스트 형태로 간략하게 출력되어있으며, 원하는 경기 번호 입력 시, 상세 기록이 출력됩니다.
	- 

3. 선수 정보
	- 기존에 등록된 선수 및 신규 선수 조회가 가능합니다.
	- 이 곳에 등록된 선수들로 경기가 진행됩니다.

4. 신규 선수 등록
	- 경기에 투입될 선수를 등록합니다.
	- 이름 > 나이 > 성별 순으로 입력해야합니다.

0. 사용 설명서
	- 프로그램 안에서 궁금한 내용은 0번에서 추가로 확인할 수 있습니다
